# CajaControl

App web para control de caja: apertura/cierre de turnos, movimientos y transferencias.

## Requisitos
- Node.js 18+ y npm

## Uso local
1) npm install
2) npm run dev
3) Abrir http://localhost:5176 (o el puerto que indique Vite)

## Build
- npm run build
- npm run preview

## Datos
- Los datos se guardan en el navegador usando localStorage.
- Exportes a Excel desde el panel Admin (todo o pesta√±a actual).
